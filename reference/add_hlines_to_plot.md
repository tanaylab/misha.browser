# Add horizontal lines to a plot

Add horizontal lines to a plot

## Usage

``` r
add_hlines_to_plot(p, panel, data, browser = NULL)
```

## Arguments

- p:

  ggplot object

- panel:

  Panel configuration

- data:

  Data frame with values for stat calculations

- browser:

  Optional browser object (needed for quantile_global)

## Value

Updated ggplot object
