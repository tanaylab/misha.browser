# Extract tracks from misha

Extract tracks from misha

## Usage

``` r
extract_tracks(tracks, region, iterator = 32, colnames = NULL)
```

## Arguments

- tracks:

  Character vector of track names/expressions

- region:

  Interval data frame

- iterator:

  Iterator value

- colnames:

  Column names for the result (defaults to tracks)

## Value

Data frame with extracted data
