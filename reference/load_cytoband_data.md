# Load cytoband data

Load cytoband data

## Usage

``` r
load_cytoband_data(cytoband_file, cytoband_track, chrom)
```

## Arguments

- cytoband_file:

  Path to cytoband file

- cytoband_track:

  Misha intervals name for cytobands

- chrom:

  Chromosome to filter to

## Value

Data frame with cytoband data
