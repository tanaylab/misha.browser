# misha.browser: Interactive Genome Browser for Misha Tracks

A configurable, interactive genome browser built on ggplot2 and
patchwork. Provides YAML-based configuration for visualization of misha
genomic tracks with support for custom transformations, faceting,
vertical line annotations, and gene models. Includes a Shiny application
for interactive exploration.
